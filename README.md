

# 📊 Telecom Customer Churn Prediction

## 📌 Project Overview
This project analyzes **customer churn in the telecom industry** using multiple approaches:
- **Data exploration in Excel**
- **SQL-based analysis**
- **Interactive dashboards in Power BI**
- **Machine Learning models** for churn prediction

The goal is to identify key factors influencing customer churn and build predictive models to help telecom companies **reduce churn rates** and improve customer retention.

---

## 📂 Repository Structure
| Folder | Description |
|--------|-------------|
| **1.Data** | Raw datasets used for analysis |
| **2.Excel_EDA** | Exploratory Data Analysis performed in Excel |
| **3.SQL_Analysis** | SQL scripts for querying and analyzing churn data |
| **4.Power BI** | Power BI dashboards for visualization and insights |
| **5.Machine_Learning** | Jupyter notebooks and ML models for churn prediction |
| **6.Images** | Supporting images and visualizations |
| **README.md.txt** | Initial documentation notes |

---

## ⚙️ Methodology
1. **Data Collection & Cleaning**  
   - Imported telecom customer data  
   - Handled missing values and inconsistencies  

2. **Exploratory Data Analysis (EDA)**  
   - Conducted descriptive statistics in Excel  
   - Identified churn patterns and correlations  

3. **SQL Analysis**  
   - Wrote queries to segment customers  
   - Analyzed churn by demographics, usage, and services  

4. **Power BI Dashboard**  
   - Built interactive dashboards  
   - Visualized churn trends and KPIs  

5. **Machine Learning Models**  
   - Implemented classification algorithms (Logistic Regression, Random Forest, etc.)  
   - Evaluated models using accuracy, precision, recall, and F1-score

## 🚀 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/nasarmohiddin-ops/Telecom_customer_churn_prediction.git
   ```
2. Navigate to the relevant folder (Excel, SQL, Power BI, ML).
3. Run Jupyter notebooks in the **Machine_Learning** folder to train and evaluate models.
4. Open Power BI files to explore interactive dashboards.

## 📈 Results
- Identified **key churn drivers** such as contract type, monthly charges, and customer service calls.  
- Built ML models with promising accuracy for predicting churn.  
- Created dashboards to help stakeholders monitor churn trends in real-time.
